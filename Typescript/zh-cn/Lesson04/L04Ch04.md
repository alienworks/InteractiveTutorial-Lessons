                        break;

                        break;
                    }
                }

我们现在已经设计完成了游戏，但在将来我们可能希望将游戏世界与其他人的联系起来。

在TypeScript NEO智能合约中，我们可以通过以下方式调用其它的合约中的方法，

```typescript
import { Address, Fixed, SmartContract } from '@neo-one/smart-contract';

interface Token {
  readonly transfer: (from: Address, to: Address, amount: Fixed<8>) => string | false;
}

const tokenAddress = Address.from('APyEx5f4Zm4oCHwFWiSTaph1fPBxZacYVR');

export class Example extends SmartContract {
  public attemptTransfer(from: Address, to: Address, amount: Fixed<8>): void {
    const smartContract = SmartContract.for<Token>(tokenAddress);
    if (smartContract.transfer(from, to, amount) === false) {
      // do something on failure
    } else {
      // do something else on success
    }
  }
}
```

首先，我们需声明另一个合约的接口，并使用`SmartContract.for<>()` 引用自身。 当调用接口中的方法之一时，我们只需要编写`contract.mothod（parameters）`。

当处理调用失败的情况时，我们通常可以使用`try/catch`，但是错误不会在智能合约的边界上传播。 这意味着我们将永远不会收到其他合约的错误。 如果抛出错误，则整个交易立刻失败。 因此，在这里，我们期望在另一个合约中的方法可返回 `false`。

### 指令

- 添加一个名为`migrate`的新方法，它以合约地址和帐户地址为参数。
- 添加一个名为 `notifyMigration`的新事件通知程序，它与`notifyGeneration`有相同的输入。
- 在方法`migrate()`中，尝试通过其地址调用另一个合约，并通过调用方法 `tokensOf()`将帐户的Aliens转移到我们的游戏中。
- 请记住通过通知程序 `notifyMigration()`通知此事件。
