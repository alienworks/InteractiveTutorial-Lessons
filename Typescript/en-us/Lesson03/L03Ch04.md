In the previous chapter, we defined two basic rules - reward and punishment. Now it's time to define more and build the story.

When Aliens go out and explore the world, they're going to go through a lot of places. For example, a road blocked by stones, an area polluted by gas and a bridge likely to collapse. We need to define some rules so that we can know what will happen.

To make the game reasonable but challenging, there are two principles,
- The event result should be based one of the Alien's attribute.
- But it needs some randomness so that the result is unpredictable.

Here the D100 is helpful. It realizes the possibility. As all these attributes of an Alien have a value from `0` to `99`, we can regard them as some possibilities and set the pass line of the event as `100`.

When an Alien is trying to pass a weak bridge, for instance, its weight decides whether the bridge will break. In the code, we are going to check,

```typescript
if (weight + this.d100() > 100)
```

If the weight value is `60`, then the possibility to cross the bridge safely is 40%. And the possibility that the D100 give a number not larger than `40` is just 40%. So if the D100 give a number not larger than `40`, then the result of `weight + this.d100() > 100` will be false, which means the bridge won't break.
